## 一、运行环境及下载代码

准备运行环境

- python3.0及以上
- neo4j 3.5.0及以上
- jdk 1.8.0

```
下载 基于医疗领域知识图谱的问答系统的代码
git clone https://github.com/zhihao-chen/QASystemOnMedicalGraph
```

## 二、搭建知识图谱

安装好neo4j然后安装脚本需要的库

```
pip install py2neo
pip install pandas
```

然后运行如下命令

```
python build_graph.py 
```

改数据比较多，运行了一个多小时
